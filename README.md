# 🕐 Digital Clock
A futuristic digital clock built with HTML, CSS, and JavaScript.
## Features
- Live time with hours, minutes, and seconds
- Displays current date, timezone, week number, and day of year
- Glowing neon blue design with scanline effect
- Updates every second in real time
## Built With
- HTML5
- CSS3
- Vanilla JavaScript
## How to Run
Clone the repo and open `index.html` in your browser.
